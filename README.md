# HODL CLUB/囤币党社区

缘起：Hodl这个单词并不是打错了，是在比特币社区中发展的一个俚语，指长期持有某种加密货币（一般指比特币）而不卖出或使用，在比特币论坛上一篇发表于2013年12月的的帖子被认为是该词的起源，其帖子主题为《I AM HODLING》。
